# PosIfGUI ソリューション概要

このリポジトリは、**東芝テック製 決済端末「A8700シリーズ」** と連携するための、画面表示確認ツールを含んだ .NET Framework Winformsベースの C# ソリューションです。


## 開発環境

- Visual Studio 2022（.NET Framework 対応）
- .NET Framework 4.7.2 以上
- Windows 10 / 11（WinForms 実行環境）
- NuGet パッケージ管理形式：`packages.config`
- 使用言語：C#

## GUI ツールについて

| アプリ名 | 実行形式 | 目的 | 自動化可否 |
|----------|----------|------|-------------|
| `PosIfGUI` | WinForms | WinFormsコンポネントベースの決済端末画面を再現・表示するUI確認ツール（いわゆる POS エミュレータ） | ❌ 手動確認 |

アプリ起動前に必ずA8700決済端末とシリアルポートで接続してからご利用ください。

## 実装範囲
現在はクレジットとイオンギフトの決済のみ実装されています。
今後、他の決済方法を追加する予定です。

## 利用の例

①端末接続 \
②本アプリ起動 \
③シリアルポートを選択（どのポートがA8700を簡単に確認するためは、接続している状態としていない状態でプルダウンメニューで確認が出来ます）\
④接続ボタン押下 \
⑤決済金額入力 \
⑥お会計ボンタンスタート \
⑦POSのフローをシムレイションスタート \
